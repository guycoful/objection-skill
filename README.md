# Objection Skill

סקיל ל-Claude Code שמטפל בהתנגדויות מכירה. מזהה מה הלקוח באמת מתכוון מאחורי "יקר לי" / "אני צריך לחשוב על זה" / "אני צריך להתייעץ" / בקשת הנחה / "מי מבטיח לי תוצאות", ונותן כיוון תגובה לפי מודל סדור.

## מה הסקיל עושה

הסקיל לא מנחש ולא ממציא. הוא לוקח את ההתנגדות שהלקוח אמר, ועושה שלושה דברים:

1. **מזהה מה הלקוח באמת מתכוון.** כי "יקר לי" זה כמעט אף פעם לא באמת על הכסף.
2. **מסביר את הטעות שרובנו עושים שם**, ולמה היא דוחפת את הלקוח החוצה.
3. **נותן כיוון תגובה לפי מודל PR**: מכילים, מבודדים את החשש האמיתי, מטפלים בו, ואז מחזירים את השיחה לתנועה.

## חמש ההתנגדויות שמכוסות

- "זה יקר לי"
- "אני צריך לחשוב על זה"
- "אני צריך להתייעץ"
- בקשת הנחה / מיקוח
- "מי מבטיח לי תוצאות ?"

## התקנה

### Claude Code

```bash
git clone https://github.com/guycoful/objection-skill.git
cp -r objection-skill ~/.claude/skills/sales-objections
```

או להתקנה ברמת פרויקט בודד:

```bash
cp -r objection-skill .claude/skills/sales-objections
```

אחרי ההעתקה, Claude Code יזהה את הסקיל אוטומטית. כדי להפעיל, פשוט תארו מצב של התנגדות, למשל: "לקוח אמר לי שזה יקר, מה לענות ?"

## איך זה שונה מסקיל NEGOTIATE

NEGOTIATE הוא יועץ משא ומתן שמלווה אותך לאורך כל השיחה. OBJECTION ממוקד ברגע אחד: כשעולה התנגדות ספציפית. השניים משלימים, לא מחליפים.

השימוש חופשי. התוכן ניתן כערך לקהילה.
